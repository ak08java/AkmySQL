                                       HAVING Cluase

SQL having clause is similar to the WHERE clause;

 either they are both used to filter row in a table based on condition 

Having clause was added to SQL because the WHERE keyword connot be used with aggregate functions.

The having clause filter the grouped rows.

The Having clause must always be followed by GROUP BY clause.

It can also contain multiple conditions filter the grouped result.

These condition seperated by various operation like AND OR etc.

Sytax
------
SELECT coulmn _name(s), function_name(argument) 
FROM table_name
WHERE condition
GROUP BY column_nmae
HAVING condition
ORDER BY column_name;

Q1) Find the number of employee working in each department as total_employee froom the table
 and the number of employee should more than one.Also sort result by total_employee
SELECT  department, COUNT(*) AS total_employee  FROM employee GROUP BY department HAVING COUNT(*) > 1 ORDER BY total_employee;

OR

SELECT department, COUNT(*) AS total_employee  FROM employee GROUP BY department HAVING total_employee > 1 ORDER BY total_employee;

FROM employee
